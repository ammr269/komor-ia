# ML Dashboard - Interface d'Analyse

Une interface web moderne pour l'analyse d'articles avec des modèles de machine learning.

## 🚀 Démarrage rapide

### Prérequis
- Node.js 18+ installé sur votre machine
- npm ou yarn

### Installation

1. **Téléchargez le projet** et extrayez-le dans un dossier
2. **Ouvrez un terminal** dans le dossier du projet
3. **Installez les dépendances** :
\`\`\`bash
npm install
\`\`\`

4. **Lancez le serveur de développement** :
\`\`\`bash
npm run dev
\`\`\`

5. **Ouvrez votre navigateur** et allez sur [http://localhost:3000](http://localhost:3000)

## 📱 Fonctionnalités

### 🔮 K-MIA - Interface de Chat
- Zone de messages scrollable
- Validation de texte (minimum 200 caractères)
- Simulation d'analyse pendant 3 secondes
- Affichage progressif des résultats de classification
- Compteur de caractères en temps réel

### 🤖 Modèles ML Disponibles
- **🌲 Random Forest** - Ensemble d'arbres de décision
- **📈 SVM** - Support Vector Machine
- **📊 Régression Logistique** - Modèle statistique simple
- **🚀 XGBoost** - Gradient boosting avancé

## 🛠️ Technologies utilisées

- **Next.js 14** - Framework React
- **Tailwind CSS** - Framework CSS utilitaire
- **React Hooks** - useState, useEffect, useRef
- **JavaScript ES6+** - Syntaxe moderne
